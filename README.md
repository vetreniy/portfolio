Данный репозиторий называется portfolio.

Он содержит в себе папки course-1, course-2, 
файлы .gitignore и README.md

В папках course-1 и course-2 содержатся папки
homeworks, practices и projects.

файл .gitignore содержит типы 
файлов, которые должен игнорировать Git

файл README.md содержит информацию о репозиторие

в папках homeworks будут размещаться 
папки с названиями формата: homework N, 
где N - номер задания.

в данных папках будет находиться домашнее задание

в папках practices будут находиться практические задания

в папках projects будут находиться проекты

когда будет известно содержимое папок projects и practices
