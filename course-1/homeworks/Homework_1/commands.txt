    git init: Инициализация нового Git-репозитория в текущей директории.

    git clone [url]: Клонирование существующего удаленного репозитория на локальный компьютер.

    git status: Показ состояния файлов в рабочей директории и индексе.

    git config --global user.name "[name]": Установка имени пользователя для всех репозиториев.

    git config --global user.email "[email]": Установка электронной почты пользователя для всех репозиториев. 

    git add [file]: Добавление изменений файла в индекс для последующего коммита.

    git add .: Добавление всех измененных файлов в текущей директории в индекс.

    git diff: Просмотр изменений, которые еще не добавлены в индекс.

    git commit -m "Сообщение коммита": Сохранение изменений из индекса в репозиторий с сообщением.

    git rm [file]: Удаление файла из индекса и рабочей директории.

    git restore [file]: Отмена изменений в файле, который не был добавлен в индекс. 

    git log: Просмотр истории коммитов.

    git show [commit-hash]: Просмотр информации о конкретном коммите.

    git reset [commit-hash]: Отмена последних коммитов. Используйте git reset --hard для полного сброса.

    git revert [commit-hash]: Создание нового коммита, который отменяет предыдущий коммит. 

    git branch: Список всех веток в репозитории.

    git branch [branch-name]: Создание новой ветки.

    git checkout [branch-name]: Переключение на другую ветку.

    git merge [branch-name]: Слияние указанной ветки с текущей. 

    git fetch: Загрузка изменений из удаленного репозитория без их слияния.

    git pull: Загрузка изменений из удаленного репозитория и их слияние с текущей локальной веткой.

    git push: Отправка локальных коммитов в удаленный репозиторий.

    git remote -v: Просмотр списка удаленных репозиториев.
